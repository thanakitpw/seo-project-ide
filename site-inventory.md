## กฎการ Internal Link

1. **Pillar Page** รับ link จากทุกบทความใน Cluster นั้น
2. **Landing Page** ให้ link ไปจากบทความ informational ที่เกี่ยวข้อง (1-2 ครั้งต่อบทความ)
3. **Blog → Blog** link ข้าม Cluster ได้ถ้าเนื้อหาเกี่ยวข้องกันจริงๆ
4. **Anchor text** ต้องอธิบายหน้าปลายทางชัดเจน ห้ามใช้ "คลิกที่นี่" หรือ "อ่านเพิ่มเติม"
5. ลิงก์ไปหน้า Service เมื่อผู้อ่านน่าจะสนใจจ้างงาน ไม่ใช่ทุกบทความ
