# Cover Image Prompter — Memory

## CI Style (Owner-Confirmed)
- Background: deep navy (#0A1628) + glowing neon cyan circuit board pattern
- Accent: neon cyan (#00B4D8), white glow
- Thai text overlay: added AFTER generation in Figma/Canva — NOT baked into AI prompt
- Brand names always in Latin characters (Claude, ChatGPT, Anthropic, Google, Meta)
- Style: professional tech corporate, 16:9
- No logos of third-party platforms (OpenAI, Google, Meta) — use abstract orbs/shapes instead

## AI & Automation Cluster — Working Formula
- Use luminous geometric orbs/nodes to represent AI entities
- Circuit board surface as the "ground plane"
- Data stream lines to show connection or flow between concepts
- Small business silhouettes (buildings) in lower third for Thai SME context
- Avoid humanoid robots and sci-fi horror vibes

## Confirmed Cover Image Sizes
- OG Image: 1200 x 630px
- Full cover: 1600 x 900px

## Prompt Structure (all tools)
[Shot type], [Subject], [Key elements], [Lighting], [Colors], [Style], [Mood], [Specs]
- Midjourney: ~100-150 words + --ar 16:9 --style raw --v 6
- DALL-E 3 / nano-banana-pro: ~80-120 words, conversational natural language
- Minimal: ~40-60 words, works across all tools
