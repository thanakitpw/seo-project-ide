# Content Brief Strategist — Recurring Patterns

## AI Tool / Product Definition Articles ("X คืออะไร")

### Structure Pattern
Used for: Claude AI, ChatGPT, n8n, Zapier, and similar AI/SaaS tool definition keywords
Search intent: Mixed Informational + Commercial Investigation

Optimal H2 sequence:
1. What is X (Featured Snippet paragraph here)
   - H3: Who made it / Company background (PAA)
   - H3: How it compares to the market leader (PAA + comparison table)
2. Versions / Plans / Tiers (if product has multiple SKUs)
3. Pricing / Free plan (PAA — always present for SaaS)
4. Thai language support (PAA — always present for Thai audience)
5. Use cases for Thai SME (unique angle — fill content gap)
6. FAQ (5 Q&A, JSON-LD)

### Comparison Table Rule
- AI tool comparison articles always have a content gap on competitor side — no one builds clear comparison tables
- Always include comparison table in the "How does X differ from Y" H3
- Minimum 4 dimensions: context/capability, free plan, Thai language, primary use case

### Word Budget per Section (Blog Post 1,200–1,500 words)
- Section 1 (definition + 2 H3s): ~300 words
- Section 2 (versions/tiers): ~200 words
- Section 3 (pricing): ~200 words
- Section 4 (Thai language): ~150 words
- Section 5 (use cases — unique angle): ~300 words max
- Section 6 (FAQ): ~150 words
- Total: ~1,300 words — fits within 1,500 word limit

### LSI Keyword Clusters for AI Tools
- Company name (e.g., Anthropic, OpenAI, Zapier)
- Safety/methodology framework if notable (e.g., Constitutional AI)
- Product tier names (e.g., Haiku/Sonnet/Opus, GPT-3.5/GPT-4)
- Technical term most commonly searched alongside (e.g., context window, LLM)
- Workflow integration term (e.g., AI Automation workflow, n8n integration)
- Thai-specific term (e.g., แชตบอต AI, ภาษาไทย)

### CTA Pattern for AI Tool Definition Articles
- Stage: Awareness/Consideration — use soft sell only
- Primary copy template: "อยากนำ [Tool] มาใช้กับธุรกิจของคุณ — ปรึกษาทีม AI Automation ของเราฟรี"
- Primary URL: /services/ai-automation
- Placement: inline after pricing section + full-width button before FAQ
- Secondary: link to AI Automation pillar page from use case section
