# n8n คืออะไร? เครื่องมือ Automation ฟรีที่ SME ไทยใช้แทน Zapier ได้

ถ้าเคยได้ยิน Zapier หรือ Make.com แล้วรู้สึกว่าราคาแพงเกินไปสำหรับธุรกิจขนาดเล็ก — n8n คืออีกตัวเลือกที่น่าสนใจมาก บทความนี้จะอธิบายตรงๆ ว่า n8n คืออะไร เหมาะกับใคร และต้องเสียเงินไหม

## n8n คืออะไร?

**n8n คือเครื่องมือสร้าง Workflow Automation แบบ open-source ที่ให้คุณเชื่อมต่อแอปและระบบต่างๆ เข้าหากัน แล้วให้มันทำงานโดยอัตโนมัติโดยไม่ต้องเขียนโค้ด** ชื่ออ่านว่า "n-eight-n" เหมือนย่อมาจาก "nodemation" — ทำงานโดยใช้ระบบ node ที่ลาก-วางเชื่อมต่อกัน แต่ละ node คือแอปหรือการทำงานหนึ่งอย่าง เช่น "รับอีเมล" → "วิเคราะห์ด้วย AI" → "บันทึกลง Google Sheets"

จุดที่ต่างจากเครื่องมืออื่น คือ n8n เป็น open-source ที่ติดตั้งบน server ตัวเองได้ฟรี ทำให้ไม่มีค่าใช้จ่ายแบบ per-task ที่กินงบเหมือน Zapier

## n8n ทำอะไรได้บ้าง?

ลองนึกถึงงานในธุรกิจที่ต้องทำซ้ำๆ ทุกวัน n8n สามารถจัดการได้เกือบทั้งหมด:

- **รับออเดอร์จาก LINE OA** แล้วบันทึกลง Google Sheets และแจ้งเตือนทีมผ่าน LINE Group อัตโนมัติ
- **ส่ง follow-up อีเมล** หลังลูกค้ากรอกฟอร์มสมัครสมาชิก โดยปรับข้อความตามข้อมูลที่กรอก
- **ดึงข้อมูล Facebook Ads** มาสรุปเป็นรายงาน Excel ส่งให้เจ้าของธุรกิจทุกวันจันทร์เช้า
- **เชื่อม AI** เช่น ChatGPT หรือ Claude มาช่วยวิเคราะห์ข้อความลูกค้าก่อนส่งต่อให้ทีม

สิ่งที่ทำให้ n8n แตกต่างคือรองรับ AI workflow เต็มรูปแบบ ใส่ LLM เข้าไปใน workflow ได้เลยโดยไม่ต้องเขียนโค้ด — ซึ่งเหมาะมากกับยุคที่ [AI Automation](/blog/ai-automation-คืออะไร) เริ่มเป็นเรื่องจำเป็นสำหรับ SME

## n8n เหมาะกับใคร?

n8n ไม่ได้เหมาะกับทุกคน ต้องพูดตรงๆ:

**เหมาะถ้า** คุณหรือทีมมี technical sense เล็กน้อย ไม่กลัวลองผิดลองถูก และต้องการ workflow ที่ซับซ้อนกว่า Zapier ทำได้ หรือต้องการประหยัดค่าใช้จ่ายระยะยาว

**ไม่เหมาะถ้า** ต้องการ setup เร็วมากโดยไม่ต้องเรียนรู้อะไรเลย หรือทีมไม่มีใครสนใจเรื่องระบบเลย — ในกรณีนี้ Zapier หรือ Make.com จะเริ่มได้ง่ายกว่า

## วิธีเริ่มต้นกับ n8n

มีสองทาง:

**ทาง 1 — n8n Cloud (ง่ายกว่า)**
สมัครที่ n8n.io แล้วใช้งานได้เลยผ่าน browser ไม่ต้องติดตั้งอะไร ทดลองใช้ฟรี 14 วัน หลังจากนั้นเริ่มต้น $20/เดือน

**ทาง 2 — Self-host (ฟรีถาวร)**
ติดตั้งบน server ตัวเองผ่าน Docker หรือ VPS ราคาถูก เช่น DigitalOcean $6/เดือน ต้องการความรู้ technical เล็กน้อย แต่ไม่มีค่า usage เลย เหมาะกับธุรกิจที่มี workflow เยอะ

สำหรับมือใหม่ แนะนำเริ่มจาก Cloud ก่อน ทำความคุ้นเคยกับระบบ node แล้วค่อยย้ายมา self-host ถ้าต้องการประหยัดค่าใช้จ่าย

## คำถามที่พบบ่อยเกี่ยวกับ n8n

### n8n ฟรีจริงไหม?

ฟรีจริง ถ้าติดตั้ง self-host เอง ไม่มีค่า operations หรือค่า task ไม่จำกัดจำนวนครั้งที่รัน workflow แต่ต้องมี server ซึ่งมีค่าใช้จ่ายราว $5-10/เดือน ถ้าใช้ n8n Cloud มีค่าบริการเริ่มต้น $20/เดือน

### n8n ต้องเขียนโค้ดไหม?

ไม่จำเป็น workflow ส่วนใหญ่ทำได้ด้วย drag-and-drop แต่ถ้าต้องการ logic พิเศษ สามารถเพิ่ม JavaScript node เข้ามาได้ ทำให้ยืดหยุ่นกว่าเครื่องมือ no-code ทั่วไป

### n8n เชื่อมกับ LINE OA ได้ไหม?

ได้ผ่าน LINE Messaging API น8n มี HTTP Request node ที่เชื่อมกับ API ใดก็ได้รวมถึง LINE, Shopee, Lazada หรือระบบ custom ของธุรกิจคุณ
